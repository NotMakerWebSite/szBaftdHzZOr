## 供应商协同系统

> [点我下载源码或在线试用](https://www.notmaker.com/detail/9d0f5343a6d145029c5655b4a6536b4c/ghb20250810) 

> 支持定制、讲解、修改、远程调试

### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（员工）、管理员。

**普通用户功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 公告管理
5. 供应商信息管理
6. 商品管理
7. 出入库管理
8. 商品进货管理
9. 商品退货管理


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 基础数据管理（公告类型管理、供应商类型管理、供应商信用等级管理、进货类型管理、退货类型管理、商品类型管理）
5. 公告管理
6. 供应商信息管理
7. 商品管理
8. 出入库管理、出入库详情管理
9. 商品进货管理
10. 商品退货管理
